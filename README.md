# 한옥 온, 습도 자동 제어 시스템
학부 졸업작품(캡스톤 디자인, 팀 프로젝트)

### 진행 기간 
- 22.09.05 ~ 23.05.22

### 역할
- 제어 소프트웨어 구현
- 제어 앱 구현

### 개발 환경
- IDE : Arduino IDE
- 앱 : MIT App inventor

### 프로젝트 설명
- 스마트폰 제어 앱으로 한옥 하드웨어 및 온, 습도 제어
- 블루투스 모듈을 통해 스마트폰, 아두이노 간 통신 연결

### 프로젝트 성과
- 온, 습도 센서 기준 값 제어 시스템 구현
- 서보모터, 릴레이, 발열패드, 팬 등 하드웨어 제어
- 제어 알고리즘 구현
- 제어 회로 구성

### 프로젝트 후 배운 점
- 협업은 서로 다른 능력의 상호작용을 통해 이뤄짐을 깨달음
- 제어 알고리즘 구현 지식 습득
